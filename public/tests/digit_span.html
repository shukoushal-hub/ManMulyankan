Digit Span Test

A standardised measure of short-term and working memory capacity. You will hear a sequence of digits displayed one at a time, then recall them in the correct order. Listen carefully — each sequence is shown once only.

Sets
2
Total Items

Set 1
Digits Forward
Sequences of 2 to 9 digits are displayed one per second. Repeat the digits back in the exact same order. Tests short-term auditory memory and attention.
8 Items · 2 Trials Each
Set 2
Digits Backward
Sequences of 2 to 8 digits are displayed one per second. Repeat the digits back in reverse order. Tests working memory and mental manipulation ability.
7 Items · 2 Trials Each
Instructions Watch each digit carefully as it appears on screen. After the full sequence has been shown, a number pad will appear — enter the digits you recall using the on-screen pad or your keyboard. The test stops automatically after two consecutive failures on any given item length.
