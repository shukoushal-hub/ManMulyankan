Trail Making Test

A clinically validated neuropsychological assessment. Complete both parts for a full cognitive profile.

Part A — Processing Speed

  • Numbers 1–25 scattered randomly on screen
  • Connect them in order: 1 → 2 → 3 → … → 25
  • All circles look identical — you must find each one
  • Measures: visual search & psychomotor speed

Part B — Executive Function

  • Numbers 1–13 and Letters A–L mixed on screen
  • Alternate: 1 → A → 2 → B → 3 → C → …
  • All circles look identical — no hints given
  • Measures: cognitive flexibility & working memory
⚠ Important: This is a self-administered digital version for educational and screening purposes only. All circles are identical — you must visually scan to find each target. Results do not constitute a clinical diagnosis.
